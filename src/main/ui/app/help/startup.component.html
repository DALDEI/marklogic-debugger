Welcome to the MarkLogic Debugger!

To get started turn on debugging above.

The next time you run your code MarkLogic will stop automatically at the first instruction.

Press the refresh arrow in the Requests pane to show all of the waiting requests. Simply click on one to start debugging.

Don't forget to turn off debugging when you are finished!
